Final Project For EECE 6045 - Intro to Quantum Computing at the University of Cincinnati. Algorithm suggested by Dr. Marc Cahay.

# Simple Example

```bash
> g++ qft.cpp -I./eigen -o qft -lm
> ./qft 
  Number of Qubits: 3
  Input State: (0-7): 0
  Target State: (0-7): 3
  Enter number of iterations: 100
> python3 plot.py
```
